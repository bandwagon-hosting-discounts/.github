## 搬瓦工是什么？先搞清楚再掏钱

搬瓦工的英文名叫 **BandwagonHost**，是加拿大 IT7 Networks 旗下的 VPS 品牌，2004 年成立，算是行业里年头很长的老牌商家了。名字之所以叫"搬瓦工"，是因为英文发音和汉语拼音"banwagong"很像，就这么叫开了。

它卖的是 **KVM 架构的 VPS**（虚拟专用服务器），最大的特点是：

- 自研 **KiwiVM Control Panel** 管理面板，操作简单，支持一键重装系统、一键迁移机房
- 支持支付宝付款，中国用户买起来很方便
- 机房覆盖美国（洛杉矶、弗里蒙特、纽约、新泽西）、日本（大阪、东京）、香港、荷兰阿姆斯特丹等地
- 高端线路有电信 CN2 GIA、联通 9929、移动 CMIN2，对国内访问速度优化明显

不过也有一个让初次找官网的人头疼的地方：**搬瓦工主域名 bandwagonhost.com 在国内已被 DNS 污染**，直接访问很可能打不开。官方提供了几个镜像站可以正常访问，比如 bwh81.net、bwh88.net、bwh89.net，内容完全一样，选哪个都行。

## 新手最常有的几个问题

在真正讲套餐之前，先把几个高频问题说清楚，省得后面看着看着又绕回来。

**问：VPS 是什么，我需要它吗？**

VPS 就是一台在远程数据中心的小型服务器，你可以用它部署网站、搭建代理、运行 24 小时挂机程序、学习 Linux 等等。如果你只是想临时用用，买个月付的就行；如果有长期需求，年付通常便宜很多。

**问：搬瓦工适合新手吗？**

相对适合。KiwiVM 面板操作门槛比较低，支付宝付款，中文资料也多，遇到问题基本都能搜到解决方案。

**问：国内用搬瓦工速度怎么样？**

这个答案很依赖你选的套餐和机房。便宜的 KVM/CN2 GT 套餐走的是普通国际线路，速度一般；CN2 GIA-E 套餐可以接入 DC6、DC9、日本软银等高速机房，对电信、联通、移动三网都有优化，速度明显好得多；香港 CN2 GIA 套餐延迟最低，但价格也最贵。
